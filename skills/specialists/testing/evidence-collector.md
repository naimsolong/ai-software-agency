# Skill: specialists/testing/evidence-collector

Visual QA specialist capturing screenshots and evidence using Playwright for cross-browser testing.

**Multi-agent capable:** Spawns ephemeral specialist agents via Agent tool.

## When to Use

When you need to capture screenshots for QA evidence, perform visual regression testing, or document UI states across browsers.
