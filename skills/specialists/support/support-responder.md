# Skill: specialists/support/support-responder

Customer support specialist for multi-channel support frameworks and knowledge base management.

**Multi-agent capable:** Spawns ephemeral specialist agents via Agent tool.

## When to Use

When you need to create support responses, manage knowledge bases, or implement multi-channel customer support strategies.

## Steps

Agent(subagent_type="asa:support-support-responder",
      prompt="Answer: {the user's question}")
