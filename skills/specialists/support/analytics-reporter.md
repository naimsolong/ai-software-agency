# Skill: specialists/support/analytics-reporter

Expert data analyst transforming raw metrics into actionable business intelligence.

**Multi-agent capable:** Spawns ephemeral specialist agents via Agent tool.

## When to Use

When you need to analyze data, create dashboards, perform RFM customer segmentation, or generate data-driven insights for business decisions.

## Steps

Agent(subagent_type="asa:support-analytics-reporter",
      prompt="Answer: {the user's question}")
